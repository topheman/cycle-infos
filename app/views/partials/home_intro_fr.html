Bienvenue

JCDecaux a implanté dans diverses villes des réseaux de vélos en libre-service (tel que le Velib' à Paris).

Accédez, en temps réel aux informations sur la disponibilité de ces vélos ainsi que des places libres pour chaque station.

Pour les développeurs

Ceci est un projet perso fait sous AngularJS, afin d'aller plus loin que la simple todo-list que nous avons tous faite. Pour plus d'informations, voir l'article sur mon blog.

Le code source est disponible sur github sous licence MIT, plus d'informations techniques dans le README.
